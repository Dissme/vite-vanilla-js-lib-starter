# 说明

## 整合的工具

- vite
- husky
- lint-staged
- vitest
- eslint
- prettier
- commitlint
- commitizen
- release-it

基本上满足了原生js-lib所需要的功能

## 使用方法

```bash
git clone https://github.com/Dissme/vite-vanilla-js-lib-starter.git
```
建议使用npm或者yarn v1安装依赖
如果使用Yarn v2安装依赖的话commitizen可能会有些问题
